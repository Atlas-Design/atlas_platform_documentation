The **Multimodal Node** is Atlas’ core image-conditioning generator.  
It **always requires both text and at least one image** as input.  
You may also provide **multiple images** simultaneously for richer conditioning.

The node combines all provided images + the text instruction and generates a new output image using the selected backend model and seed.

---

## 1. Overview

The Multimodal Node performs image-guided generation, where:

- **Images define**: structure, layout, objects, style cues, or references  
- **Text defines**: what transformation, modification, or extraction should occur  

The output is an AI-generated image that respects both the visual references and the instruction.

---

## 2. Inputs

### Required:
- **Text prompt**  
- **At least one image**  

### Optional:
- **Multiple images** (e.g., replaceable objects, reference styles, guides)

The node merges all image inputs before applying the instruction.

---

## 3. Backends

You may choose from multiple backend models.  
Each backend produces:

- different fidelity  
- different artistic interpretation  
- different structural preservation  
- different detail behavior  

Switching backend = switching to a different AI model.

## 6. Capabilities Summary

The Multimodal Node supports:

### **Image-Guided Generation**
- interpret and transform scenes
- restyle environments
- change themes (fantasy, futuristic, damaged, aged)

### **Object-Level Editing**
- add new objects (from reference images)
- replace selected objects
- remove objects while preserving layout
- isolate individual items cleanly

### **Asset Preparation**
- produce isolated items on neutral backgrounds
- prepare clean references for Image-to-3D workflows

### **Multiple Input Conditioning**
- use several images at once  
- combine scene + object references  
- merge style and structural hints from multiple sources  

### **Controlled Variation**
- backend selection for model diversity  
- seed control for multiple variations  

---

## 7. Node Summary

The **Multimodal Node** is Atlas’ flexible, image-conditioned generation engine.  
It:

- **requires text + image(s)**  
- supports multiple reference images  
- generates structured scene edits  
- isolates objects for asset pipelines  
- transforms renders while keeping layout stable  
- provides high variation through backends and seeds  

It is the foundation for workflows involving **editing, restyling, isolating, transforming, or augmenting images** across Atlas.
